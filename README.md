# ED_Lista02.06
## Inverter Vetor: Recursiva

Este desafio consiste em criar uma aplicação, em Java, que troque a posição de seus valores.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A inversão deve ser realizada por uma função recursiva;
3. A inversão deve seguir: N troca de posição com (tamanhoVetor-N);
4. A função deve ser testada com um vetor de 10 e outro de 11 posições;
5. Declarar, via comentário, a condição de parada da função.
```
